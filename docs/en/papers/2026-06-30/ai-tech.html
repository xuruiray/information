ComMem: Complementary Memory Systems for Test-Time Adaptation of Vision-Language Models arXiv 人工智能

The paper proposes ComMem, a complementary memory system to enhance test-time adaptation of vision-language models in dynamic environments, improving robustness.

Read more → Counterfactual Residual Data Augmentation for Regression arXiv 机器学习
