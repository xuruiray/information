Developer Tools Briefing

Developer tools: MAVEN framework improves agent tool-calling generalization; new context management for long-horizon tasks; MiniMax M3 arrives on Vercel; Braintrust accelerates experiments with Codex; NumLeak benchmark reveals model memorization risks.

MAVEN: Improving Generalization in Agentic Tool Calling arXiv 人工智能

MAVEN framework significantly improves LLM generalization across tool-calling environments via combined reasoning strategies, state preservation, and cross-domain tool coordination.

Read more → Learning Agent-Compatible Context Management for Long-Horizon Tasks arXiv 人工智能

Proposes a learning-based context management method to help LLM agents avoid context degradation and reasoning failures in long-horizon tasks like web search.

Read more → MiniMax M3 on AI Gateway Vercel 博客

MiniMax M3 is the first model with 1M-token context window and native multimodality, based on sparse attention, excelling at software engineering and agentic web browsing.

Read more → NumLeak: Public Numeric Benchmarks as Latent Labels in Foundation Models arXiv 机器学习
