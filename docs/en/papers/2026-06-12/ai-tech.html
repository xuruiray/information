Frontier Models Briefing

Key frontier papers cover ToolSense for auditing parametric tool knowledge, Okara running 120k CMO agents on Vercel, TrajGenAgent for human mobility generation, Arbor's tree-search cognition layer, and strategic decision support for AI agents, advancing tool retrieval, generation, and decision-making.

ToolSense: A Diagnostic Framework for Auditing Parametric Tool Knowledge in LLMs arXiv 人工智能

LLMs deployed as agents face a tool-retrieval bottleneck. ToolSense introduces a diagnostic framework that encodes each tool as a virtual token to audit parametric tool knowledge and capture specialized semantics.

Read more → How Okara runs CMO agents for 120,000 companies on Vercel Vercel 博客

Okara processes 4 billion tokens daily on Vercel, running an AI CMO with 8 sub-agents to drive marketing growth for 120,000+ businesses, with same-day model availability.

Read more → TrajGenAgent: A Hierarchical LLM Agent for Human Mobility Trajectory Generation arXiv 人工智能

Human mobility data is costly to collect. TrajGenAgent uses a hierarchical LLM agent to generate realistic synthetic trajectories, outperforming prompt-based methods.

Read more → Arbor: Tree Search as a Cognition Layer for Autonomous Agents arXiv 人工智能

Arbor is a multi-agent framework that introduces structured tree search as a cognition layer, maintaining an explicit search tree for hypothesis exploration in large action spaces.

Read more → Strategic Decision Support for AI Agents arXiv 人工智能

In modern agentic systems, the traditional decision-support role reverses: AI agents act for users while humans become support mechanisms. The paper examines reliability challenges of this reversal.

Research & Products Briefing

This subsection features five works: lie detector evaluation, an AGI definition alignment framework, multimodal peer-review attacks/defenses, restless bandits with imperfect feedback, and physics-constrained generative models for semiconductor manufacturing.

"Did you lie?" Evaluating Lie Detectors across Model Scale and Belief-Verified Model Organisms arXiv 人工智能

The work builds testbeds where models verifiably believe the opposite of what they say, for evaluating lie detectors. Existing trained model organisms often fail this verification.

Read more → Definitional alignment before capability alignment: a Design-Science framework for adjudicating claims about AGI arXiv 人工智能

Claims about AGI existence often rely on differing definitions. This paper proposes a design-science framework to align definitions first, adjudicating claims consistently.

Read more → Does AI Reviewer See the Full Picture? Attacking and Defending Multimodal Peer Review arXiv 计算语言学

Integrating multimodal LLMs into peer review introduces adversarial risks. This study examines attacks that manipulate figures in papers and proposes defenses.

Read more → Restless bandits with imperfect binary feedback: PCL-indexability analysis and computation arXiv 机器学习

Motivated by spectrum access with sensing errors, this work develops a PCL-based analysis framework for restless bandits with imperfect binary feedback, establishing indexability.

Read more → Physics-informed generative AI for semiconductor manufacturing: Enforcing hard physical constraints in generative models by construction arXiv 机器学习

Generative models for physical systems must satisfy hard constraints. Using semiconductor manufacturing as a test case, this work enforces physical laws by construction in generative models.
