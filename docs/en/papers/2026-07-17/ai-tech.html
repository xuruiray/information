CANDI: Contextual Alignment for Niche Domains Question Answering arXiv 计算语言学

Proposes CANDI, a contextual alignment framework to enhance LLM question answering in specialized domains like medical diagnosis and financial advisory. Experiments show significant improvements over traditional models in custom benchmarks, yielding more accurate and contextually relevant outputs.

Fine-tunes LLaMA 3 (8B) via a two-stage pipeline of supervised fine-tuning and knowledge distillation into an efficient RAG reranker. It achieves high accuracy while significantly reducing inference costs, suitable for real-time deployment.

Proposes a context-augmented prompting framework that integrates graph topological cues to improve zero-shot molecular property prediction from SMILES strings in small language models. The modular approach is transferable and applicable to drug discovery.

Evaluates state-of-the-art LLMs on bidirectional Korean-Braille translation using a human-annotated dataset. Despite expected high performance, models show significant structural accessibility failures with poor translation quality.

Research & Governance

Research & Governance Subsection Briefing

This subsection includes five studies: point-in-time language models to avoid lookahead bias in financial backtesting; learning safe agent behavior from human preferences and world models; G-SHARE for structured diagnosis of human-factor events in nuclear plants; mechanistic explanation of belief-reality separation in LLMs; interventional grounding audits for chain-of-thought premise dependency.

Scaling Point-in-Time Language Models arXiv 计算语言学

Proposes point-in-time language models trained only on text up to a cutoff date, eliminating lookahead bias. Experiments show significant reduction in forward-looking bias in financial backtests and causal inference in social sciences, improving result validity.

Read more → Learning Safe Agent Behaviour from Human Preferences and Justifications via World Models arXiv 人工智能

Trains safe agent policies via world models using human preferences and justifications in settings with unknown dynamics and no reward function. Demonstrates effectiveness and safety in safety-critical environments.

Read more → G-SHARE: A Guideline-Based Structured Reasoning Framework for Human-Factor Event Diagnosis arXiv 计算语言学

Proposes G-SHARE, combining operational guidelines with structured reasoning to improve accuracy and interpretability in diagnosing human-factor events in nuclear power plants. Significantly outperforms traditional methods and one-shot LLM queries.

Read more → Belief-reality separation lives in routing over a shared value slot in language models arXiv 计算语言学

Mechanistic interpretability reveals two separate mechanisms at two positions that enable belief-reality separation in LLMs: a general layer routes to a shared value slot, and a specific layer fine-tunes the output.

Read more → Interventional Grounding Audits: Black-Box Premise-Dependency Tests for LLM Chain-of-Thought via Predicate Substitution arXiv 人工智能

Proposes interventional grounding audits, a black-box test that substitutes predicates in chain-of-thought premises to check if LLMs genuinely depend on stated premises. Experiments reveal that seemingly sound reasoning often suffers from premise dependency failures.
