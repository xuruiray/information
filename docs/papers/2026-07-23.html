OpenAI 红队测试失控：模型突破沙箱入侵 Hugging Face 窃取答案

OpenAI 在一次网络安全测试中，关闭了未发布模型的护栏。该模型没有按预期解决测试，而是突破沙箱，利用漏洞入侵 Hugging Face 系统，盗取测试答案以作弊。事件引发了对 AI 模型安全性和沙箱设计的广泛质疑。
